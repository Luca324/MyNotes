# Архитектура проекта MyNotes

## Общее описание

MyNotes - это мобильное приложение для заметок с рекурсивной иерархической структурой. Основная фича - возможность создавать темы (папки) и подтемы любой глубины, а также заметки внутри каждой темы.

## Технологический стек

- **Framework**: React Native с Expo
- **Навигация**: Expo Router (file-based routing)
- **База данных**: SQLite (expo-sqlite)
- **Язык**: TypeScript + JavaScript
- **Версия Expo**: ~53.0.23
- **Версия React**: 19.0.0
- **Версия React Native**: 0.79.5

## Основные концепции

### 1. Рекурсивная структура тем (Topics)

Темы могут иметь родительские темы (`parent_id`), что позволяет создавать иерархию любой глубины:
- Тема уровня 0 (корневая) - `parent_id = null`
- Подтема уровня 1 - `parent_id = <id корневой темы>`
- Подтема уровня 2 - `parent_id = <id подтемы уровня 1>`
- И так далее...

### 2. Заметки (Notes)

Заметки привязаны к конкретной теме через `topic_id`. Каждая заметка может иметь:
- `title` - заголовок (опционально)
- `content` - содержимое
- `order_index` - порядок сортировки

### 3. Вкладки (Tabs)

Вкладки - это закрепленные темы для быстрого доступа. Пользователь может закрепить любую тему как вкладку.

## Паттерны и подходы

- **File-based routing**: Используется Expo Router для навигации
- **Context API**: AppContext для глобального состояния (вкладки)
- **Custom Hooks**: useNotes, useModal для переиспользования логики
- **Service Layer**: databaseService.ts для абстракции работы с БД
- **TypeScript**: Типизация основных сущностей (Topic, Note, Tab)

## Цветовые схемы

Проект использует функцию `getDepthColor()` из `colorSchemes.js` для визуального отображения глубины вложенности тем.
